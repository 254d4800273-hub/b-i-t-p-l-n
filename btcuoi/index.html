Trang Debug Code

1. Nguyên văn đoạn mã lỗi từ đề bài:
function checkLoad(totalCredits) {
let msg = (totalCredits > 15) ? "Đủ tải" : "Chưa đủ tải";
document.getElementById("status").value(msg);
}
2. Đoạn mã đã sửa lỗi hoàn chỉnh kèm chú thích giải thích: 
function checkLoad(totalCredits) {
    let msg = (totalCredits > 15) ? "Đủ tải" : "Chưa đủ tải";
    
    /* SỬA LỖI TẠI ĐÂY:
       - Lỗi cũ: .value(msg) -> Coi .value như một hàm/phương thức là sai cú pháp.
       - Sửa lại: .value = msg -> gán trực tiếp giá trị cho thuộc tính 'value' (nếu là thẻ input).
       - Lưu ý thêm: Nếu phần tử có id="status" là một thẻ như <div>, <p>, <span> 
         thì ta phải dùng .innerText = msg hoặc .innerHTML = msg thay vì .value.
    */
    document.getElementById("status").value = msg; 
}
